# My personal website
A (lovely) website powered by [Hugo](https://gohugo.io) \
[www.jeorje.net](https://www.jeorje.net)

## Comandi utili
> Testing in locale
- hugo server -D 

> Builda il sito in /public
- hugo

> Nuovo post da modello
- hugo new --kind post <name>
